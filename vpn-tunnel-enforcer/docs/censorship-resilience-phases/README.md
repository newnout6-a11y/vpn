## Главные инварианты продукта

- Основной туннель обязан запускаться даже если обновление правил, подписки или
  внешние проверки временно недоступны.
- Для rule-set и runtime-ресурсов всегда должен оставаться локальный fallback.
- Любая новая stealth-функция должна иметь диагностику; никаких “магических”
  переключателей без объяснения состояния.
- Alpha-only возможности upstream нельзя включать по умолчанию для всех.
- До появления нормальной абстракции ядра проект остаётся sing-box-first.

## Когда весь трек можно считать завершённым

Трек завершён только когда в приложении есть:

- обновлённый и наблюдаемый sing-box runtime;
- управляемые rule-set каналы с last-known-good кэшем;
- расширенный импорт и запуск современных протоколов;
- понятные stealth-пресеты с диагностикой;
- отдельная политика для control-plane загрузок;
- проверенный экспериментальный путь для stealth UDP-туннеля;
- чистая граница ядра для возможной поддержки mihomo.
